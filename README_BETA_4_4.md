# Quiz Reviewer • Beta 4.4

A student-friendly quiz reviewer with instant feedback, progress tracking, dark mode, accessibility settings, and structure-based chemistry naming practice.

## What changed in Beta 4.4

- Cleaner, softer home screen with no futuristic/grid-style background.
- Chemistry naming and identification items are forced into multiple-choice format so students can practice recognition without typing.
- Structure-based items show the structure first; names, SMILES, formulas, explanations, and sources appear only after reveal/review.
- True/False choices stay ordered as True then False.
- Matching items still hide internal table letters and focus on the meaning of each choice.
- Kept elapsed timer, score tracker, settings, dark mode, Study Coach, and GitHub Pages workflow.
