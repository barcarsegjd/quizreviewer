# Quiz Reviewer • Beta 4.5

Beta 4.5 cleans up the Study Coach provider menu.

## Changes
- Study Coach now shows only two choices: **Deepseek** and **Gemini**.
- Removed visible **Gemini API**, **DeepSeek API**, **Auto**, and **Offline** choices from the dropdown.
- The app still keeps a built-in fallback message if the online coach is unavailable.
- Retains Beta 4.4 improvements: multiple-choice naming/identification, MolView-style structures, elapsed timer, progress tracker, accessibility settings, dark mode, instant feedback, and auto-next.

## Update steps
Copy `src/App.jsx` into your project, then commit the change with GitHub Desktop.
