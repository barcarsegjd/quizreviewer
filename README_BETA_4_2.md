# Quiz Reviewer Beta 4.2

A student-friendly quiz reviewer with instant feedback, progress tracking, accessibility settings, and structure-based naming practice.

## Beta 4.2 update

- Added MolView-style molecular structure images for chemistry nomenclature and identification items.
- Functional-group recognition items now show structures instead of relying only on condensed formulas.
- Names, SMILES, and formulas stay hidden while answering and appear only after reveal/review.
- Retained all quizzes, progress tracker, settings, dark mode, study coach, and GitHub Pages deployment workflow.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

Use the included `.github/workflows/deploy.yml` and set GitHub Pages source to GitHub Actions.
