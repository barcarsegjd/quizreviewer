# Quiz Reviewer • Beta 4.3

Student-friendly quiz reviewer with instant feedback, progress tracking, structure-based chemistry naming, and an elapsed-time timer during every quiz attempt.

## New in Beta 4.3

- Added an elapsed-time timer while taking a quiz.
- Timer appears in the top-right question navigator.
- Timer appears in the quiz stat cards as `Elapsed time`.
- Results page now shows the total time spent for the attempt.
- Progress Tracker still saves time spent for every submitted attempt.

## Install

Copy `src/App.jsx` into your project `src` folder.

If you are coming from before Beta 4.2, also copy `public/molview` into your project `public` folder.
